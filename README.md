# Airline Loyalty & Revenue Analytics (SQL Project)

## Overview
This project uses SQL to analyze airline customer behavior, loyalty engagement,
marketing promotions, and revenue performance. It is designed to simulate
real-world analytics and decision-support work in an airline environment.

## Objectives
- Analyze loyalty customer engagement
- Measure revenue performance and booking trends
- Evaluate marketing campaign effectiveness
- Support ad-hoc and scenario-based decision making

## Tools Used
- SQL (MySQL / SQL Server / Oracle compatible)

## Key Insights
- Identified high-value loyalty customers
- Measured impact of promotions on bookings
- Analyzed revenue trends across routes and time

## Use Cases
- Associate Analyst / Data Analyst roles
- Airline & travel analytics
- Business decision support
